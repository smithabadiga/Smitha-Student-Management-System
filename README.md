v# Student Management System

## Description

Student Management System is a console-based Java application that helps manage student records. It allows users to add, view, search, update, and delete student information using a menu-driven interface.

## Features

* Add Student
* View All Students
* Search Student by ID
* Update Student Details
* Delete Student
* Menu-Driven Program
* Uses ArrayList for Data Storage

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Scanner Class

## Project Structure

* Student.java
* StudentManager.java
* Main.java

## How to Run

1. Compile all Java files.
2. Run Main.java.
3. Select options from the menu.
4. Perform CRUD operations on student records.

## Sample Menu

1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit
